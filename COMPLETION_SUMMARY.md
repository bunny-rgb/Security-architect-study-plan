## 🛠 Technical Architecture

### Frontend Stack
- **Next.js 14** (App Router, TypeScript)
- **React 18** (Client components, hooks)
- **Tailwind CSS v4** (Custom theme, animations)
- **Mobile-first** design approach

### Backend Stack
- **Hono** (Lightweight framework)
- **Cloudflare Workers** (Edge runtime)
- **Cloudflare D1** (SQLite database)
- **Wrangler** (Dev & deployment)

### Infrastructure
- **PM2**: Process management (2 services)
  - `webapp-api` on port 3001 (Hono/Wrangler)
  - `webapp-ui` on port 3000 (Next.js standalone)
- **Proxy Setup**: Next.js rewrites API calls to port 3001
- **Hot Reload**: Live updates during development

## 🔧 Development Commands

```bash
# Start both services
pm2 start ecosystem.config.cjs

# Check status
pm2 list

# View logs
pm2 logs --nostream

# Restart services
pm2 restart all

# Stop services
pm2 stop all

# Database operations
npm run db:migrate:local  # Apply migrations
npm run db:seed:local     # Load content
npm run db:reset          # Reset & reseed

# Build commands
npm run build:api         # Build Hono API
npm run build:frontend    # Build Next.js UI
npm run build            # Build both

# Clean up
npm run clean-port        # Kill port 3000
```
